## 二、竞赛选择指南（跨学科）

### 面向用户：不知道选哪个竞赛的学生

#### 选择决策框架

```
第一步：兴趣筛选
├── 喜欢纯逻辑推理和证明？→ 数学
├── 喜欢理解物理现象和建模型？→ 物理
├── 喜欢做实验和微观世界？→ 化学
├── 喜欢生命科学和自然观察？→ 生物
└── 喜欢编程和解决问题？→ 信息学

第二步：能力匹配
├── 数学天赋极强（年级数学前三）→ 优先考虑数学
├── 物理+数学都强 → 物理（性价比更高）
├── 记忆力强+化学直觉好 → 化学
├── 记忆力强+对生命科学感兴趣 → 生物
└── 逻辑清晰+有编程基础 → 信息学

第三步：投入产出评估
├── 目标国集保送 → 数学/信息学（纯靠天赋，风险大）
├── 目标强基破格 → 化学/生物（省一/国二的难度相对较低）
├── 目标高考提分 → 物理/化学（对高考直接帮助最大）
└── 时间有限 → 只选一科，不要多线作战

第四步：学校资源
├── 学校有强竞赛教练和培训体系 → 优先选学校的优势科
├── 学校没有竞赛体系 → 生物/化学（自学相对容易）
└── 需要报班 → 考虑家庭经济投入
```

#### 能否同时学两科竞赛？

| 组合 | 可行性 | 建议阶段 | 冲突程度 |
|:---|:---|:---|:---|
| 物理+数学 | 困难 | 只建议数学天才 | 高（时间冲突严重） |
| 物理+化学 | 可行 | 高一开始物理，高二加化学 | 中（知识有交叉） |
| 化学+生物 | 较易 | 高一开始化学，高二加生物 | 低（知识互补） |
| 信息学+数学 | 困难 | 不建议 | 高（都是高强度逻辑训练） |
| 信息学+物理 | 可行 | 视个人时间而定 | 中 |

**核心建议**：大多数学生**只选一科**，精力集中效果最好。如果非要选两科，建议选**化学+生物**（难度较低、知识互补）或**物理+化学**（对高考帮助最大）。

## 三、物理竞赛（CPhO）详细规划

> 物理竞赛作为五科竞赛的**详细范本**，以下规划可直接参照到其他四科。

### 3.1 竞赛 vs 强基 vs 高考：三层难度定位

| 维度 | 高考 | 强基计划 | 物理竞赛（CPhO） |
|:---|:---|:---|:---|
| 目标用户 | 全体高中生 | 前5%-10% | 前1%-2% |
| 知识范围 | 高考大纲 | 高考+大学先修 | 竞赛大纲（≈大学物理专业大一+大二） |
| 数学要求 | 高中数学 | 微积分基础 | 微积分+微分方程+复变函数+矢量分析 |
| 题目难度 | 标准场景熟练应用 | 非标准场景分析 | 多模型深度耦合+原创性分析 |
| 考试形式 | 高考（6月） | 高考+校测 | 预赛→复赛→决赛（9-10月） |
| 周期 | 高三一年冲刺 | 高三一年准备 | 通常需要2-3年系统准备 |
