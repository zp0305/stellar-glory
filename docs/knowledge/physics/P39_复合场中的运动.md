## 学科本质

> **学科本质**：复合场是重力场+电场+磁场的叠加——粒子受力分析与运动判断的关键在于"先分力、再合运动"，不同力做功特性决定能量处理方式。

> **常见迷思**：
> - "复合场问题无从下手"——关键先判断各力做功情况和运动性质，重力做功与路径无关、电场力做功与路径无关、洛伦兹力永不做功。

## 叙事正文

### 情境锚定（100-150字）

电视机显像管中的电子束，要在屏幕上精确扫描出画面，需要同时受到电场的加速和磁场的偏转控制。质谱仪利用电场加速、磁场偏转来分离不同质量的同位素。回旋加速器中，粒子在电场中加速、在磁场中偏转，一次次循环直到获得极高能量。这些装置的核心都是"复合场"——粒子同时处于两种或三种场中，运动规律比单一场复杂得多，但也因此能实现精密控制。

### 困惑预设（150-200字）

很多学生面对复合场问题时觉得"无从下手"，其实是因为没有建立正确的分析框架。常见错误有两种：一是把所有力一股脑代入牛顿第二定律列方程，不分析哪些力存在、哪些力为零；二是混淆"速度选择器"和"质谱仪"等标准模型的原理。核心思路是：先判断**哪些力存在**（重力是否可忽略？有没有电场？有没有磁场？），再判断**运动性质**（是否做圆周运动？是否做类平抛？是否做直线运动？），最后选择**解题方法**（动力学方程还是能量守恒）。

### 实验/现象呈现（200-250字)

**速度选择器**：两平行板间加匀强电场E（方向竖直向下），同时加垂直纸面向里的匀强磁场B。当带正电粒子以某一速度v水平射入时，电场力$F_E = qE$向下，洛伦兹力$F_B = qvB$向上。当$qE = qvB$时，粒子直线通过，此时$v = E/B$。大于或小于这个速度的粒子都会偏转被挡板吸收。

**质谱仪**：粒子先经电场加速（$qU = \frac{1}{2}mv^2$），再进入磁场做匀速圆周运动（$qvB = mv^2/r$）。由两式得$r = \frac{1}{B}\sqrt{\frac{2mU}{q}}$——质量不同的粒子偏转半径不同，打到感光底片上的位置不同，从而实现质量分离。

**回旋加速器**：粒子在D形盒间的电场中加速，在D形盒内的磁场中做半圆周运动。每加速一次，速度增大，半径增大，但周期不变$T = 2\pi m/(qB)$，所以交变电场的频率保持恒定。

### 概念涌现（150-200字)

复合场问题中，三种力的做功特性完全不同，这是选择解题方法的关键依据：

| 力 | 做功特性 | 适用守恒定律 |
|:---|:---|:---|
| 重力 | 与路径无关 | 机械能守恒（无电场时） |
| 电场力 | 与路径无关 | 动能定理（含电势能） |
| 洛伦兹力 | 永不做功 | 不参与能量变化 |

**核心结论**：洛伦兹力只改变速度方向，不改变速度大小，所以**动能定理中不出现洛伦兹力**。当只有重力和电场力做功时，用动能定理或能量守恒往往比牛顿定律更高效。

### 推导叙事（300-400字)

**1. 速度选择器的速度条件**

带电粒子以速度v进入正交的E和B场：

- 电场力：$F_E = qE$（方向由E决定）
- 洛伦兹力：$F_B = qvB$（方向由左手定则确定）

当粒子匀速直线通过时：$qE = qvB$，解得：

$$v = \frac{E}{B}$$

注意：这个结果与粒子的电荷量q和质量m**都无关**，只取决于E和B的比值。

**2. 质谱仪的偏转半径**

粒子经电压U加速后进入磁场：

- 加速阶段：$qU = \frac{1}{2}mv^2$，得$v = \sqrt{\frac{2qU}{m}}$
- 偏转阶段：$qvB = \frac{mv^2}{r}$，得$r = \frac{mv}{qB}$

代入v：$r = \frac{1}{B}\sqrt{\frac{2mU}{q}}$

结论：同位素（q相同、m不同）的偏转半径$r \propto \sqrt{m}$。

**3. 回旋加速器的最大动能**

D形盒最大半径为R时，粒子获得的最大速度：

$v_{max} = \frac{qBR}{m}$

最大动能：$E_{k,max} = \frac{1}{2}mv_{max}^2 = \frac{q^2B^2R^2}{2m}$

注意：最大动能与加速电压U**无关**，只取决于B和R。

### 迁移应用（150-200字)

**霍尔效应**：通电导体在磁场中，载流子受洛伦兹力偏转，在导体两侧积累电荷形成电势差（霍尔电压），可用于测量磁场强度。

**电磁流量计**：导电流体在管道中流动，外加磁场，流体中的正负离子受洛伦兹力偏转，在管道两侧产生电势差，电势差大小与流速成正比，从而实现流量测量。

**磁流体发电机**：高温等离子体高速通过磁场，洛伦兹力使正负电荷分离到两侧电极，直接将动能转化为电能，无需机械转动部件。

## 前置知识检测（3道题）

**Q1（检验P38带电粒子在磁场中的运动）**：单选 | 难度1  
题目：带电粒子在匀强磁场中做匀速圆周运动，下列说法正确的是  
A. 速度越大，周期越大  
B. 磁感应强度越大，周期越大  
C. 周期与速度和半径都无关  
D. 粒子动能越大，周期越大  
答案：C  
解析：$T = 2\pi m/(qB)$，周期只与m、q、B有关，与速度和半径无关。

**Q2（检验P31+P38→P39衔接）**：单选 | 难度2  
题目：带电粒子以速度v垂直进入正交的匀强电场E和匀强磁场B中，粒子恰好做匀速直线运动。若保持B不变，将E增大为原来的2倍，粒子将  
A. 仍做匀速直线运动  
B. 向电场力方向偏转  
C. 向洛伦兹力方向偏转  
D. 做圆周运动  
答案：B  
解析：原来$qE = qvB$平衡，E加倍后$qE > qvB$，粒子向电场力方向偏转。

**Q3（概念辨析）**：单选 | 难度2  
题目：关于复合场中带电粒子的运动，下列说法正确的是  
A. 洛伦兹力对粒子做正功，使粒子加速  
B. 在电场和磁场的复合场中，粒子动能一定增大  
C. 速度选择器只能选择特定速度的粒子，与粒子电荷量和质量无关  
D. 回旋加速器中粒子的最大动能与加速电压成正比  
答案：C  
解析：A错，洛伦兹力永不做功；B错，动能是否增大取决于是否有电场力或重力做功；C对，$v = E/B$与q、m无关；D错，$E_{k,max} = q^2B^2R^2/(2m)$与U无关。

## 分层变形

### B级例题（直接套公式）

**B级例题1**：速度选择器  
题目：速度选择器中$E = 3×10^4 V/m$，$B = 0.1T$，能直线通过选择器的粒子速度是多少？  
解：$v = E/B = 3×10^4/0.1 = 3×10^5 m/s$  
核心点：速度选择器的通过速度$v = E/B$，与粒子种类无关。

**B级例题2**：质谱仪偏转半径  
题目：质谱仪中，$^{12}C^+$离子和$^{14}C^+$离子经相同电压U=1000V加速后进入同一磁场$B = 0.5T$，求两者的偏转半径之比。$m_{12} = 12u$，$m_{14} = 14u$，$1u = 1.66×10^{-27}kg$。  
解：$r = \frac{1}{B}\sqrt{\frac{2mU}{q}}$，q相同、U相同、B相同  
$r_1/r_2 = \sqrt{m_1/m_2} = \sqrt{12/14} = \sqrt{6/7} \approx 0.926$  
核心点：同位素的偏转半径之比等于质量平方根之比。

### J级例题（综合分析）

**J级例题1**：电场+磁场复合  
题目：在竖直向下的匀强电场$E = 4×10^3 V/m$和垂直纸面向里的匀强磁场$B = 0.5T$中，一个质量$m = 1×10^{-3}kg$、电荷量$q = +2×10^{-3}C$的小球从静止释放，求小球的最大速度。（取$g = 10 m/s^2$）  
解：  
受力分析：重力$mg = 0.01N$向下，电场力$qE = 8N$向下，洛伦兹力$F_B = qvB$向上（速度向右时）。  
开始时合力向下，小球加速。随着v增大，洛伦兹力增大。  
当$qE + mg = qvB$时合力为零，速度最大：  
$v_{max} = (qE + mg)/(qB) = (8 + 0.01)/(2×10^{-3}×0.5) = 8.01/0.001 = 8010 m/s$  
核心点：最大速度出现在合力为零的时刻，此时洛伦兹力+其他力平衡。

### T级例题（拓展挑战）

**T级例题1**：多过程——回旋加速器  
题目：回旋加速器D形盒半径$R = 0.5m$，磁感应强度$B = 1.6T$，加速电压$U = 2×10^4V$。求质子（$m = 1.67×10^{-27}kg$，$q = 1.6×10^{-19}C$）获得的最大动能和加速次数。  
解：  
最大动能：$E_{k,max} = \frac{q^2B^2R^2}{2m} = \frac{(1.6×10^{-19})^2×(1.6)^2×(0.5)^2}{2×1.67×10^{-27}} = \frac{2.56×10^{-38}×2.56×0.25}{3.34×10^{-27}} = \frac{1.638×10^{-38}}{3.34×10^{-27}} \approx 4.91×10^{-12}J \approx 30.7MeV$  
每次加速获得动能$ΔE_k = qU = 2×10^4×1.6×10^{-19} = 3.2×10^{-15}J$  
加速次数：$n = E_{k,max}/(qU) = 4.91×10^{-12}/3.2×10^{-15} \approx 1534$次  
核心点：回旋加速器的最大动能与加速电压无关，但加速次数与电压有关——电压越低，需要的加速次数越多。

**T级例题2**：图像分析  
题目：一带电粒子从静止开始，先经电压$U_1$加速，再垂直进入磁感应强度为$B_1$的磁场做圆周运动，测得圆周运动周期为$T_1$。若将加速电压加倍、磁感应强度加倍，新周期$T_2$是多少？  
解：  
周期$T = 2\pi m/(qB)$，与加速电压无关。  
$T_1 = 2\pi m/(qB_1)$，$T_2 = 2\pi m/(q×2B_1)$  
$T_2/T_1 = B_1/(2B_1) = 1/2$，$T_2 = T_1/2$  
核心点：圆周运动周期只取决于m、q、B，与速度无关。B加倍则周期减半。

**T级例题3**：实验探究  
题目：在速度选择器实验中，已知$E = 2.0×10^4V/m$，$B = 0.2T$。现有三种粒子：质子（$m_p = 1.67×10^{-27}kg$）、氘核（$m_D = 3.34×10^{-27}kg$）、α粒子（$m_α = 6.64×10^{-27}kg$），都带正电且电荷量分别为$e$、$e$、$2e$。求每种粒子能通过选择器的速度，并说明选择器的"筛选"特性。  
解：  
三种粒子的通过速度都是$v = E/B = 2.0×10^4/0.2 = 1.0×10^5 m/s$  
但实际进入的粒子如果初速度不是这个值就会被偏转挡住。  
核心点：选择器的本质是"速度过滤"，不是"质量过滤"或"电荷过滤"——同速度的不同粒子都能通过。质谱仪在后面加磁场才能实现质量分离。
