## 学科本质

> **学科本质**：交变电流是"大小和方向随时间周期性变化的电流"——正弦交流电是其中最重要的。

> **常见迷思**：
> - "交流电的有效值就是平均值"——有效值是平方平均的平方根（均方根）

## 叙事正文

### 情境锚定（100-150字）

家里插座输出的标称"220V"到底是什么含义？万用表测出的是峰值还是别的什么量？

实际上，我国家用交流电的电压随时间按正弦规律变化，峰值约为311V，方向每秒改变100次。但电器铭牌上写的不是311V，而是220V——这个"220V"是交流电的**有效值**。

那么：有效值和峰值之间有什么关系？它和平均值是一回事吗？

### 困惑预设（150-200字）

很多同学第一次接触交流电时，会产生一个很自然的直觉："交流电大小在变化，取个平均值不就行了？"

这个想法看似合理，却隐藏着一个根本性的误区。考虑一个极端情况：电流在前半个周期为 $+10A$，后半个周期为 $-10A$，一个完整周期的**算术平均值**恰好是零——可显然不能说"没有电流通过"。平均值忽略了方向变化的抵消效应，更关键的是忽略了功率与电流的平方成正比这一物理事实。

要正确衡量交流电的做功能力，必须从**热效应等效**出发：让交流电和直流电通过同一个电阻，在相同时间内产生相同热量，该直流电的值就是交流电的**有效值**。它不是简单的平均，而是**平方平均的平方根**（均方根）。

### 实验/现象呈现（200-250字）

**实验：线圈在匀强磁场中转动产生交流电**

将一个匝数为 $N$、面积为 $S$ 的矩形线圈放置在磁感应强度为 $B$ 的匀强磁场中，线圈以角速度 $\omega$ 绕垂直于磁场的中心轴匀速转动。线圈两端通过滑环和电刷连接到示波器上。

观察到的现象：
1. **示波器显示正弦波形**：当线圈匀速转动时，示波器上呈现出完美的正弦（余弦）曲线，说明感应电动势随时间按正弦规律变化。
2. **峰值出现在线圈平面与磁场平行时**：此时磁通量变化率最大，感应电动势达到峰值 $E_m = NB\omega S$。
3. **零值出现在线圈平面与磁场垂直时**：此时磁通量最大，但变化率为零，感应电动势为零。

这就是正弦交流电的产生过程，也是所有后续分析（有效值、功率、变压器原理）的物理基础。

### 概念涌现（150-200字）

从实验中涌现出以下核心概念：

**1. 瞬时值**：任意时刻 $t$ 的电动势 $e = E_m \sin(\omega t)$，电流 $i = I_m \sin(\omega t)$。它们描述交流电在每一瞬间的状态。

**2. 峰值（最大值）**：$E_m = NB\omega S$，是瞬时值能达到的最大值。它由线圈参数和转动角速度决定。

**3. 有效值（均方根值）**：基于热效应等效定义。对于正弦交流电，有效值与峰值的关系为 $E = \dfrac{E_m}{\sqrt{2}}$。

**4. 周期与频率**：$T = \dfrac{2\pi}{\omega}$，$f = \dfrac{1}{T}$。我国市电 $f = 50\text{ Hz}$，$T = 0.02\text{ s}$，$\omega = 100\pi\text{ rad/s}$。

**5. 相位与初相**：$e = E_m \sin(\omega t + \varphi_0)$，其中 $\varphi_0$ 为初相位，描述 $t=0$ 时刻线圈的位置状态。

### 推导叙事（300-400字）

**一、推导瞬时值表达式 $e = E_m \sin(\omega t)$**

设矩形线圈匝数为 $N$、面积为 $S$，匀强磁场磁感应强度为 $B$，线圈绕垂直于磁场的轴以角速度 $\omega$ 匀速转动。

取 $t = 0$ 时线圈平面与磁场垂直（中性面位置），经过时间 $t$，线圈从中性面转过的角度 $\theta = \omega t$。此时线圈平面的法线方向与磁场方向的夹角即为 $\theta$。

穿过线圈的磁通量：
$$\Phi = BS\cos\theta = BS\cos(\omega t)$$

由法拉第电磁感应定律，感应电动势的瞬时值：
$$e = N \frac{d\Phi}{dt} = N \cdot BS\omega \cdot \sin(\omega t)$$

令峰值 $E_m = NB\omega S$，则：
$$\boxed{e = E_m \sin(\omega t)}$$

类似地，若电路总电阻为 $R$，则电流瞬时值 $i = \dfrac{e}{R} = I_m \sin(\omega t)$，其中 $I_m = \dfrac{E_m}{R}$。

**二、推导有效值 $E = E_m / \sqrt{2}$**

有效值的定义：让交流电通过电阻 $R$，在一个周期 $T$ 内产生的热量，与某一直流电通过同一电阻在相同时间内产生的热量相等，该直流电的值即为有效值。

交流电在一个周期内的热量：
$$Q_{\text{交}} = \int_0^T \frac{e^2}{R}\, dt = \int_0^T \frac{E_m^2 \sin^2(\omega t)}{R}\, dt$$

直流电产生的热量：
$$Q_{\text{直}} = \frac{E^2}{R} \cdot T$$

令 $Q_{\text{交}} = Q_{\text{直}}$，利用 $\displaystyle\int_0^T \sin^2(\omega t)\, dt = \frac{T}{2}$，可得：
$$\frac{E_m^2}{R} \cdot \frac{T}{2} = \frac{E^2}{R} \cdot T \implies E = \frac{E_m}{\sqrt{2}}$$

同理：$I = \dfrac{I_m}{\sqrt{2}}$，$U = \dfrac{U_m}{\sqrt{2}}$。

**注意**：$E = E_m/\sqrt{2}$ 仅对正弦（余弦）交流电成立，非正弦交流电需重新积分计算。

### 迁移应用（150-200字）

**1. 家用交流电的计算**：我国市电频率 $f = 50\text{ Hz}$，电压有效值 $U = 220\text{ V}$，则峰值 $U_m = 220\sqrt{2} \approx 311\text{ V}$。普通白炽灯两端电压实际在 $-311\text{ V}$ 到 $+311\text{ V}$ 之间变化，但灯泡亮度取决于功率的平均效果，即由有效值决定。

**2. 交流电压表的读数**：交流电压表和电流表测量的是有效值，不是峰值，也不是平均值。这是仪器设计的物理依据——有效值直接反映了交流电的做功能力。

**3. 电容器的耐压选择**：电容器接入交流电路时，两端承受的峰值电压可达 $311\text{ V}$，因此额定电压至少应大于 $311\text{ V}$，而非仅大于 $220\text{ V}$。若选 $250\text{ V}$ 的电容，看似"大于220V"，实际会被击穿。

## 前置知识检测（3道题）

**Q1（检验P40电磁感应）**：单选 | 难度1
题目：一个匝数 $N = 100$、面积 $S = 0.01\text{ m}^2$ 的线圈在磁感应强度 $B = 0.5\text{ T}$ 的匀强磁场中，从中性面开始以 $\omega = 10\pi\text{ rad/s}$ 匀速转动，感应电动势的峰值为
A. $15.7\text{ V}$
B. $50\text{ V}$
C. $31.4\text{ V}$
D. $5.0\text{ V}$
答案：A
解析：$E_m = NB\omega S = 100 \times 0.5 \times 10\pi \times 0.01 = 5\pi \approx 15.7\text{ V}$

**Q2（检验P40→P42衔接）**：单选 | 难度1
题目：线圈在匀强磁场中匀速转动时，下列哪个位置磁通量的变化率最大？
A. 线圈平面与磁场平行时
B. 线圈平面与磁场垂直时（中性面）
C. 线圈平面与磁场成30°角时
D. 线圈平面与磁场成45°角时
答案：A
解析：磁通量 $\Phi = BS\cos\theta$，变化率 $\frac{d\Phi}{dt} = -BS\omega\sin\theta$，当 $\theta = 90°$（线圈平面与磁场平行）时，$\sin\theta = 1$，变化率最大，此时感应电动势达峰值。

**Q3（概念辨析）**：单选 | 难度2
题目：关于交流电的有效值，下列说法正确的是
A. 有效值等于交流电在一个周期内的算术平均值
B. 有效值等于峰值的一半
C. 有效值是根据热效应等效来定义的，正弦交流电有效值为峰值的 $1/\sqrt{2}$
D. 任何波形的交流电有效值都等于峰值的 $1/\sqrt{2}$
答案：C
解析：有效值通过热效应等效定义，正弦交流电的有效值 $E = E_m/\sqrt{2}$。算术平均值不能衡量做功能力（甚至可能为零）；$E = E_m/\sqrt{2}$ 仅对正弦交流电成立。

## 分层变形

### B级变形（2题）

**例题1**：直接套公式
题目：某正弦交流电的电动势峰值 $E_m = 311\text{ V}$，频率 $f = 50\text{ Hz}$。写出电动势的瞬时值表达式，并求有效值。
解：
- 角速度 $\omega = 2\pi f = 100\pi\text{ rad/s}$
- 瞬时值 $e = 311\sin(100\pi t)\text{ V}$
- 有效值 $E = \dfrac{311}{\sqrt{2}} \approx 220\text{ V}$
核心点：$e = E_m\sin(\omega t)$，$E = E_m/\sqrt{2}$

**例题2**：直接套公式
题目：一个标有"220V 100W"的白炽灯接在 $u = 311\sin(100\pi t)\text{ V}$ 的交流电源上，求通过灯泡的电流有效值和灯泡正常工作时的电阻。
解：
- 电压有效值 $U = \dfrac{311}{\sqrt{2}} = 220\text{ V}$（与额定电压一致，正常工作）
- 电流有效值 $I = \dfrac{P}{U} = \dfrac{100}{220} \approx 0.455\text{ A}$
- 电阻 $R = \dfrac{U^2}{P} = \dfrac{220^2}{100} = 484\,\Omega$
核心点：交流电路中功率计算用有效值，$P = UI = I^2R = U^2/R$

---

### J级变形（1题）

**例题1**：综合分析
题目：一个 $N = 200$ 匝、面积 $S = 0.02\text{ m}^2$ 的矩形线圈在磁感应强度 $B = 0.4\text{ T}$ 的匀强磁场中绕垂直于磁场的轴匀速转动，角速度 $\omega = 50\pi\text{ rad/s}$，线圈总电阻 $r = 2\,\Omega$，外接电阻 $R = 18\,\Omega$。求：(1) 感应电动势的峰值；(2) 电路中电流的有效值；(3) 电阻 $R$ 上消耗的功率。
解：
- (1) $E_m = NB\omega S = 200 \times 0.4 \times 50\pi \times 0.02 = 80\pi \approx 251.3\text{ V}$
- (2) 电流峰值 $I_m = \dfrac{E_m}{R + r} = \dfrac{80\pi}{18 + 2} = 4\pi \approx 12.57\text{ A}$，电流有效值 $I = \dfrac{I_m}{\sqrt{2}} = \dfrac{4\pi}{\sqrt{2}} \approx 8.89\text{ A}$
- (3) $R$ 上消耗的功率 $P_R = I^2 R = \left(\dfrac{4\pi}{\sqrt{2}}\right)^2 \times 18 = 32\pi^2 \times 0.5 \times 18 = 288\pi^2 \approx 2842\text{ W}$
核心点：从线圈参数求峰值，再用有效值计算功率，注意区分峰值与有效值

### T级变形（3题）

**例题1**：多过程分析
题目：一个正弦交流发电机（$E_m = 311\text{ V}$，$f = 50\text{ Hz}$）通过理想变压器给远距离输电线供电，变压器原线圈 $n_1 = 1000$ 匝，副线圈 $n_2 = 100$ 匝。输电线总电阻 $R_{\text{线}} = 10\,\Omega$，负载电阻 $R_L = 90\,\Omega$。求：(1) 副线圈两端电压的有效值；(2) 输电线上损耗的功率。
解：
- (1) 原线圈电压有效值 $U_1 = E_m/\sqrt{2} = 311/\sqrt{2} = 220\text{ V}$
- 由变压器变压比 $\dfrac{U_2}{U_1} = \dfrac{n_2}{n_1}$，得 $U_2 = 220 \times \dfrac{100}{1000} = 22\text{ V}$
- (2) 回路总电阻 $R_{\text{总}} = R_{\text{线}} + R_L = 10 + 90 = 100\,\Omega$
- 回路电流 $I_2 = \dfrac{U_2}{R_{\text{总}}} = \dfrac{22}{100} = 0.22\text{ A}$
- 输电线损耗功率 $P_{\text{损}} = I_2^2 R_{\text{线}} = 0.22^2 \times 10 = 0.484\text{ W}$
核心点：有效值全程使用，变压器只变电压/电流不变功率（理想变压器）

**例题2**：图像分析
题目：一个交流电压随时间变化——电压在前半个周期为 $+100\text{ V}$ 恒定，后半个周期为 $-100\text{ V}$ 恒定，周期 $T = 0.02\text{ s}$。求该交流电的电压有效值。
解：
- 一个周期内产生的热量（取电阻 $R$）：
$$Q = \frac{100^2}{R} \cdot \frac{T}{2} + \frac{100^2}{R} \cdot \frac{T}{2} = \frac{10000}{R} \cdot T$$
- 直流电产生的热量：$Q' = \dfrac{U^2}{R} \cdot T$
- 令 $Q = Q'$，得 $U^2 = 10000$，$U = 100\text{ V}$
- 该方波的有效值恰好等于其峰值，**而非**峰值的 $1/\sqrt{2}$
核心点：有效值必须从定义出发积分计算，$E = E_m/\sqrt{2}$ 仅对正弦波成立

**例题3**：实验探究
题目：为了测量交流电的有效值，某同学设计了如下实验：将一个"6V 3W"的小灯泡分别接在直流电源和交流电源上，调节直流电源使小灯泡亮度与接交流电源时相同，此时直流电压表读数为 $6.0\text{ V}$。已知交流电源输出正弦交流电，求：(1) 交流电的有效值；(2) 交流电的峰值；(3) 该实验的物理原理。
解：
- (1) 由有效值的定义（热效应等效），灯泡亮度相同说明做功能力相同，因此交流电的有效值就等于此时直流电压表的读数：$U = 6.0\text{ V}$
- (2) 峰值 $U_m = \sqrt{2}\,U = 6\sqrt{2} \approx 8.49\text{ V}$
- (3) 原理：有效值定义——"让交流电和直流电通过同一电阻，若在相同时间内产生的热量相等，则该直流电的值即为交流电的有效值"。灯泡亮度由实际消耗功率决定，亮度相同即功率相同、热量产生速率相同。
核心点：用热效应等效理解有效值的操作定义

## 公式卡片

### 核心公式

| 公式 | 名称 | 含义 |
|:---|:---|:---|
| $e = E_m\sin(\omega t)$ | 正弦交流电瞬时电动势 | $E_m = NB\omega S$ 为峰值，$\omega = 2\pi f$ 为角频率 |
| $i = I_m\sin(\omega t)$ | 正弦交流电瞬时电流 | $I_m = E_m / R_{\text{总}}$ 为电流峰值 |
| $E = \dfrac{E_m}{\sqrt{2}}$ | 有效值与峰值的关系（正弦交流电） | 有效值由热效应等效定义 |
| $I = \dfrac{I_m}{\sqrt{2}},\quad U = \dfrac{U_m}{\sqrt{2}}$ | 电流、电压的有效值 | 功率计算必须使用有效值 |
| $T = \dfrac{2\pi}{\omega} = \dfrac{1}{f}$ | 周期、频率、角频率关系 | 我国市电 $f = 50\text{ Hz}$，$T = 0.02\text{ s}$ |
