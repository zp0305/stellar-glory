## 学科本质

> **学科本质**：超重/失重描述的是"视重"（支持力）与"实重"（重力）的关系——加速度方向决定。

> **常见迷思**：
> - "失重就是没有重力"——失重时重力仍在，只是支持力变小
> - "超重就是重力变大"——重力不变，是支持力变大
> - "在电梯里体重称的读数就是重力"——那是支持力，不是重力

## 叙事正文

### 情境锚定（100-150字）

你站在体重秤上，电梯加速上升时，读数变大；电梯加速下降时，读数变小。

你的重力变了吗？没有！变化的是体重秤对你的**支持力**。

这个现象如何用物理解释？"超重"和"失重"到底是什么？

### 困惑预设（150-200字）

很多学生认为"失重就是没有重力"。

实际上：
- 宇航员在空间站"失重"，但仍然受到地球引力（重力）
- 失重只是因为支持力小于重力
- 完全失重时，支持力为零，但重力仍在

另一个困惑："电梯里体重增加，是不是重力变大了？"——不是，重力由地球决定，不会变。

### 实验/现象呈现（200-250字）

用体重秤实验验证超重与失重：

在电梯里放体重秤，记录不同运动状态的读数：

| 电梯状态 | 读数(N) | 相对静止(50N) |
|:---|:---|:---|
| 静止 | 50 | 正常 |
| 加速上升 | 55 | 超重 |
| 匀速上升 | 50 | 正常 |
| 减速上升 | 45 | 失重 |
| 加速下降 | 45 | 失重 |
| 匀速下降 | 50 | 正常 |
| 减速下降 | 55 | 超重 |

**规律**：加速上升（a向上）→超重；加速下降（a向下）→失重

### 概念涌现（150-200字）

从实验提炼超重与失重：

**超重**：当物体有**向上加速度**时，支持力大于重力
$$N = m(g + a)$$

**失重**：当物体有**向下加速度**时，支持力小于重力
$$N = m(g - a)$$

**完全失重**：当$a = g$时，$N = 0$
$$N = m(g - g) = 0$$

关键：判断超重/失重只看**加速度方向**，与速度方向无关。

### 推导叙事（300-400字）

超重与失重如何用牛顿第二定律推导？

**1. 受力分析**
物体受两个力：重力$G = mg$（向下）、支持力$N$（向上）

**2. 列方程**（设向上为正方向）
$$N - mg = ma$$

**3. 求解**
- 当$a > 0$（向上加速）：$N = m(g + a) > mg$ → 超重
- 当$a < 0$（向下加速）：$N = m(g + a) < mg$ → 失重
- 当$a = -g$：$N = 0$ → 完全失重

**4. 理解"视重"和"实重"**
- 实重：$G = mg$（由地球决定，恒定）
- 视重：$N$（体重秤读数）
- 超重：视重 > 实重
- 失重：视重 < 实重

### 迁移应用（150-200字）

超重与失重在生活中和科研中都有重要应用：

**航天员训练**：离心机旋转产生向心加速度，使人感受数倍重力——模拟火箭发射时的超重状态。

**蹦极**：下降过程中加速阶段是失重，最高点后弹力绳拉人进入超重阶段。

**微重力实验**：飞机俯冲时产生约30秒失重，用于科学实验。

## 前置知识检测（3道题）

**Q1（检验P09牛顿第二定律）**：单选 | 难度1  
题目：质量50kg的人站在电梯里的体重秤上，电梯加速上升时加速度2m/s²，秤的读数是（g=10m/s²）  
A. 500N  
B. 600N  
C. 400N  
D. 550N  
答案：B  
解析：$N = m(g+a) = 50 \times (10+2) = 600N$

**Q2（检验P09→P10衔接）**：单选 | 难度1  
题目：关于超重和失重的说法，正确的是  
A. 超重就是重力变大  
B. 失重就是没有重力  
C. 超重时加速度向上  
D. 失重时加速度向上  
答案：C  
解析：超重时物体有向上加速度，失重时有向下加速度

**Q3（概念辨析）**：简答 | 难度1  
题目：宇航员在空间站里"飘起来"是没有重力吗？  
答案：不是。空间站里的重力约为地面重力的90%，宇航员"飘"是因为他们处于完全失重状态（$N=0$），而非没有重力。  
解析：失重是支持力为零，不是重力为零

## 分层变形

### B级变形（2题）

**例题1**：直接套用公式  
题目：质量60kg的人，电梯以2m/s²加速上升，求体重秤读数（g=10m/s²）。  
解：$N = m(g+a) = 60 \times 12 = 720N$  
核心点：超重公式$N = m(g+a)$

**例题2**：直接套用公式  
题目：质量50kg的人，电梯以3m/s²加速下降，求体重秤读数（g=10m/s²）。  
解：$a向下=3$，$N = m(g-a) = 50 \times 7 = 350N$  
核心点：失重公式$N = m(g-a)$

---

### J级变形（1题）

**例题1**：多过程分析  
题目：质量50kg的人站在电梯里的体重秤上，电梯从静止开始：先以2m/s²加速上升3秒，然后匀速上升5秒，最后以3m/s²减速直到停止。求各阶段体重秤读数。  
解：  
① 加速上升：$N_1 = 50 \times (10+2) = 600N$（超重）  
② 匀速：$N_2 = 50 \times 10 = 500N$（正常）  
③ 减速上升：$a向下=3$，$N_3 = 50 \times (10-3) = 350N$（失重）  
核心点：加速度方向决定超重/失重

---

### T级变形（3题）

**例题1**：完全失重分析  
题目：电梯钢索突然断裂，电梯做自由落体运动。求电梯内体重秤的读数。  
解：$a = -g$，$N = m(g + a) = m(g - g) = 0$（完全失重）  
核心点：自由落体是完全失重

**例题2**：斜面问题  
题目：质量m的物体在倾角θ的斜面上加速下滑，加速度$a = g\sin\theta$。求斜面支持力。  
解：竖直方向列方程：$mg - N = ma_y = ma\sin\theta = mg\sin^2\theta$，所以$N = mg(1-\sin^2\theta) = mg\cos\theta$（小于重力，是部分失重）
核心点：斜面加速下滑时竖直方向有加速度分量$a_y = g\sin^2\theta$，属于部分失重

**例题3**：实验探究  
题目：设计实验测量超重和失重时加速度，写出原理和步骤。  
原理：$N = m(g+a)$，测出$N$和$m$可求$a$  
步骤：① 用体重秤测"视重" $N$ ② 计算$a = N/m - g$  
多次测量验证  
核心点：用超重/失重原理反推加速度

## 公式卡片

### 核心公式

| 公式 | 名称 | 适用条件 |
|:---|:---|:---|
| $N = m(g + a)$ | 超重公式 | a向上 |
| $N = m(g - a)$ | 失重公式 | a向下 |
| $N = 0$ | 完全失重 | a = g向下 |

### 变量说明

| 符号 | 含义 | 单位 |

| $m$ | 质量 | kg |
| $g$ | 重力加速度 | m/s² |
| $a$ | 加速度 | m/s² |
