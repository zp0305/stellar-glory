## 叙事正文

### 情境锚定（100-150字）

三个实验定律，分别描述三种特殊过程。

但实际气体状态变化往往同时涉及p、V、T三个量的变化。

能否用一个方程描述所有情况？

### 困惑预设（150-200字）

很多学生记不住标准状况的数据。

实际上：
- 标准状况：$T = 273.15K$，$p = 1atm = 101325Pa$
- 标准摩尔体积：$V_m = 22.4L/mol$（近似值）

### 概念涌现（150-200字)

理想气体状态方程：

**方程**：
$$\frac{p_1V_1}{T_1} = \frac{p_2V_2}{T_2}$$

或
$$pV = nRT$$

其中$R = 8.31 J/(mol \cdot K)$是气体常数

### 推导叙事（300-400字)

**1. 从三个实验定律推导**
由玻意耳、查理、盖吕萨克定律综合推导

**2. 一般形式**
$$\frac{pV}{T} = nR$$

**3. 适用条件**
理想气体是真实气体的近似模型：
- 分子间距足够大
- 分子间除碰撞外无相互作用

### 迁移应用（150-200字)

应用实例：

**气象学**：大气模型。

**热机**：卡诺循环。

**呼吸系统**：肺泡气体交换。

## 分层变形

### B级变形（2题）

**例题1**：直接计算  
题目：2mol气体，压强p=1×10⁵Pa，温度T=300K，求体积。  
解：$V = nRT/p = 2 \times 8.31 \times 300 / 10^5 = 0.05 m^3 = 50L$  
核心点：$pV = nRT$

**例题2**：直接计算  
题目：气体从状态1(p1=2atm, V1=3L, T1=300K)变到状态2(p2=1atm, V2=6L)，求T2。  
解：$p_1V_1/T_1 = p_2V_2/T_2$，$T_2 = 300K$  
核心点：比例计算

---

### J级变形（1题）

**例题1**：多过程分析  
题目：气体先等压加热（V1→V2），再等温压缩（p2→p3），求终态。  
解：分别用盖吕萨克和玻意耳定律  
核心点：多过程

---

### T级变形（3题）

**例题1**：微观分析  
题目：从分子动理论推导理想气体状态方程。  
解：$p = \frac{1}{3}n\overline{E_k}$，结合$\overline{E_k} = \frac{3}{2}kT$  
核心点：微观推导

**例题2**：图像分析  
题目：在p-V图上画等温线，分析做功和内能变化。  
解：等温线是双曲线，$\Delta U = 0$，$W = Q$  
核心点：热力学第一定律

**例题3**：实验探究  
题目：设计实验验证理想气体状态方程。  
原理：$pV/T = nR$  
步骤：测p、V、T，计算nR  
核心点：用实验验证
