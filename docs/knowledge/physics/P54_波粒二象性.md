## 学科本质

> **学科本质**：光既有波动性（干涉衍射）又有粒子性（光电效应）——这不仅是光的性质，也是所有微观粒子的性质。

> **常见迷思**：
> - "光既是波又是粒子"——不是"既是...又是"，而是在不同实验中表现出不同侧面

---

## 叙事正文

### 情境锚定（100-150字）

光可以发生干涉——说明光是波。

光可以从金属打出电子——说明光是粒子。

光究竟是什么？

### 困惑预设（150-200字）

很多学生认为"光既是波又是粒子"。

实际上：
- 不是同时是两者
- 而是在不同实验中表现不同侧面
- 这叫**波粒二象性**

### 核心公式（200-250字)

**光电效应**：
$$E = h\nu - W$$

**德布罗意波长**：
$$\lambda = \frac{h}{p}$$

其中$h = 6.63 \times 10^{-34} J\cdot s$是普朗克常数

### 推导叙事（300-400字)

**1. 波动性**
干涉、衍射证明光是电磁波

**2. 粒子性**
光电效应证明光粒子性

**3. 物质波**
电子、中子等也有波动性
$$\lambda = \frac{h}{mv}$$

### 迁移应用（150-200字)

应用实例：

**光电效应**：太阳能电池

**电子显微镜**：利用电子波粒二象性

**光谱分析**：物质鉴定

## 前置知识检测（3道题）

**Q1（检验P52物理光学）**：单选 | 难度1  
题目：关于光电效应，说法正确的是  
A. 任意频率光都能打出电子  
B. 光电子数目与光强成正比  
C. 光电子能量与频率成正比  
D. 光电子能量与光强无关  
答案：B  
解析：光电子数目与光强成正比

**Q2（检验P52→P54衔接）**：单选 | 难度1  
题目：电子动量p=1.65×10⁻²⁴kg·m/s，德布罗意波长是  
A. 0.4nm  
B. 4nm  
C. 40nm  
D. 400nm  
答案：A  
解析：λ=h/p=4×10⁻¹⁰m=0.4nm

**Q3（概念辨析）**：单选 | 难度2  
题目：关于波粒二象性，下列说法正确的是  
A. 光子既有波动性又有粒子性，可以同时观测到两种属性  
B. 宏观物体没有波粒二象性  
C. 大量光子表现出波动性，少量光子表现出粒子性  
D. 电子只具有粒子性，不具有波动性  
答案：C  
解析：单个光子通过双缝后落在屏上是随机的一个点（粒子性），大量光子积累形成干涉条纹（波动性），这是概率波的核心含义。A错因为两种属性不会同时被观测到；B错因为一切物体都有波粒二象性，只是宏观物体德布罗意波长极短无法探测；D错，电子衍射实验已证实电子的波动性。

## 分层变形

### B级例题（直接套公式）

**B级例题1**：光电效应方程  
题目：用波长λ=200nm的紫外线照射金属钾（逸出功W₀=2.25eV），光电子最大初动能是多少？  
解：$E = hc/λ = (6.63×10^{-34} × 3×10^8)/(200×10^{-9}) = 9.945×10^{-19}J = 6.22eV$  
$E_k = E - W_0 = 6.22 - 2.25 = 3.97eV$  
核心点：$h\nu$是入射光子能量，$W_0$是逸出功，$E_k$是最大初动能。

**B级例题2**：德布罗意波长  
题目：电子经100V电压加速后，德布罗意波长是多少？（$m_e = 9.11×10^{-31}kg$）  
解：$E_k = eU = 100eV = 1.6×10^{-17}J$，$p = \sqrt{2m_eE_k} = \sqrt{2×9.11×10^{-31}×1.6×10^{-17}} = 5.40×10^{-24}kg·m/s$  
$\lambda = h/p = 6.63×10^{-34}/5.40×10^{-24} = 1.23×10^{-10}m = 0.123nm$  
核心点：100V加速电子的德布罗意波长与X射线波长同量级，可用于晶体衍射实验。

### J级例题（综合分析）

**J级例题1**：截止频率+截止电压  
题目：某金属截止频率$ν_0 = 6×10^{14}Hz$，用$ν = 1×10^{15}Hz$的光照射，求光电子最大初动能和截止电压。  
解：$E_k = h(ν - ν_0) = 6.63×10^{-34} × (1×10^{15} - 6×10^{14}) = 6.63×10^{-34} × 4×10^{14} = 2.65×10^{-19}J = 1.66eV$  
截止电压：$U_c = E_k/e = 1.66V$  
核心点：截止电压是反向电压增大到光电流恰好为零时的电压值。

### T级例题（拓展挑战）

**T级例题1**：多过程——不同频率光照射对比  
题目：用频率分别为$ν_1 = 2ν_0$和$ν_2 = 3ν_0$的光照射同一种金属（$ν_0$为截止频率），产生的光电子最大初动能之比是多少？  
解：$E_{k1} = h(2ν_0 - ν_0) = hν_0$，$E_{k2} = h(3ν_0 - ν_0) = 2hν_0$  
比值：$E_{k1}/E_{k2} = 1/2$  
核心点：最大初动能与$(ν - ν_0)$成正比，不是与$ν$成正比。

**T级例题2**：实验——密立根光电效应实验  
题目：密立根用不同频率的光照射钠表面，测得截止电压$U_c$与频率$ν$的数据关系图是一条直线，斜率为k。求普朗克常量h的表达式。  
解：$eU_c = hν - W_0$，即$U_c = (h/e)ν - W_0/e$  
斜率$k = h/e$，所以$h = ke$  
核心点：密立根实验精确测定了直线的斜率，结合已知电子电荷e，首次精确测定了h值，验证了爱因斯坦光电效应方程。

**T级例题3**：概念辨析——不确定关系  
题目：关于不确定关系$Δx·Δp ≥ h/4π$，下列理解正确的是  
A. 粒子的位置和动量可以同时精确测量，只是仪器精度不够  
B. 不确定关系来源于测量对粒子的干扰  
C. 微观粒子的波粒二象性决定了不可能同时具有确定的位置和动量  
D. 宏观物体也受不确定关系限制，只是效应太小察觉不到  
答案：C  
解析：不确定关系是波粒二象性的数学表达，不是测量技术限制（A、B错）。D对但不如C准确——C指出了物理本质。高中阶段选择C。
