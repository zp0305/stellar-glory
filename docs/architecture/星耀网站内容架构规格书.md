# 星耀网站内容架构规格书

> **定位**：本文是星耀网站的**前端架构规格书**，面向网站制作方（开发者/前端/产品经理）。
> **职责**：定义网站有哪些页面、每个页面由什么数据组成、页面之间的关系、交互规格。
> **不包含**：内容怎么写、风格要求、质量标准（这些属于《星耀内容模板》的职责）。
> **版本**：v1.0
> **日期**：2026-04-25
> **上游文档**：《总纲v9.1》定义"为什么做"，《星耀内容模板v1.2》定义"内容怎么生产"，本文定义"网站怎么呈现"。

---

## 一、网站总览

### 1.1 产品定位

星耀是面向年级前10%-15%高中生的"认知图谱+解题套路"理解型学习系统。核心理念：**不只给公式，给"怎么想"**。

### 1.2 内容来源三层架构

```
┌─────────────────────────────────────────────┐
│  第一层：内容生成总纲（"宪法"）                │
│  定义：目标用户、能力模型、内容标准、文件边界    │
├─────────────────────────────────────────────┤
│  第二层：星耀内容模板（"施工图"）              │
│  定义：每种内容形态的填写字段和JSON结构         │
│  输出：Markdown（人读）+ JSON（网站消费）       │
├─────────────────────────────────────────────┤
│  第三层：本文档（"建筑图"）                    │
│  定义：网站页面结构、路由、数据模型、交互规格    │
│  消费：第二层产出的JSON数据                     │
└─────────────────────────────────────────────┘
```

**关键原则**：
- 第三层（本文档）**只实现**第一、二层明确定义的功能，不自行增加教育理论功能
- 数据格式由第二层（内容模板）定义，网站按格式渲染即可
- 如有歧义，以第一层（总纲）为最终裁决

### 1.3 六科内容规模

| 学科 | 知识节点 | 模型/问题类型 | 分析范式 | 思维方法 |
|:---|:---:|:---:|:---:|:---:|
| 物理 | 56 (P01-P56) | 43 (P-M01-M43) | 90 (P-R01-R90) | 7 (P-TM01-TM07) |
| 数学 | ~75 (K01-K~75) | 58 (K-F1~F11等) | ~85 (K-F01-F~85) | 9 (K-TM01-TM09) |
| 化学 | 57 (C01-C57) | 48 (C-M01-M48) | ~70 (C-F01-F~70) | 7 (C-TM01-TM07) |
| 生物 | 52 (B01-B52) | 48 (B-M01-M48) | ~60 (B-F01-F~60) | 6 (B-TM01-TM06) |
| 语文 | 43 (Y01-Y43) | 29 (Y-C01-C29) | 45 (Y-F01-F45) | 6 (Y-TM01-TM06) |
| 英语 | 49 (E01-E49) | 52 (E-M01-M52) | 53 (E-F01-F53) | 6 (E-TM01-TM06) |

> **数学特殊说明**：数学无传统"模型"层，中间层为"问题类型"（58个），用板块字母+编号编码（如F3=函数与导数第3个问题类型）。

## 二、导航结构

### 2.1 顶部栏

```
├── Logo + 平台名
├── 学科切换（物理/数学/化学/生物/语文/英语）
├── 搜索（知识节点/模型/范式/题目）
└── 个人中心
```

### 2.2 左侧栏

```
├── 首页（仪表盘/进度总览）
├── 知识节点（章节树 → 板块 → 小节 → 节点）
├── 模型详解（按板块分组，或按知识节点聚合）
├── 思维方法（独立入口，按学科核心思维方法列表）
├── 分析范式（独立入口，按模型分组）
├── 题库（按模型索引，五维标签筛选）
├── 公式库（搜索+收藏）
├── 学习进度
└── 学科指南
```

### 2.3 路由表

| 路由 | 页面 | 说明 |
|:---|:---|:---|
| `/{subject}/methods/{id}` | 思维方法卡片页 | 如 `/physics/methods/TM01` |
| `/{subject}/concepts/{id}` | 知识节点详解页 | 如 `/physics/concepts/K09` |
| `/{subject}/models/{id}` | 模型详解页 | 如 `/physics/models/M05` |
| `/{subject}/paradigms/{id}` | 分析范式页 | 如 `/physics/paradigms/R09` |
| `/{subject}/questions` | 题库页 | 练习中心，含筛选 |
| `/{subject}/guide` | 学科指南页 | 认知论型指南 |

其中 `{subject}` 取值：`physics` / `math` / `chemistry` / `biology` / `chinese` / `english`

## 三、内容形态与页面结构

### 3.1 四种核心内容页面

星耀每个学科的内容由**四种核心内容形态** + **题库** + **学科指南**组成：

```
思维方法卡片页 ── 最抽象层（/{subject}/methods）
     ↓ 链接
知识节点详解页 ── 核心概念层（/{subject}/concepts）
     ↓ 链接
模型详解页 ── 场景方法层（/{subject}/models）
     ↓ 链接
分析范式页 ── 思维路径层（/{subject}/paradigms）
     ↓ 链接
题库页 ── 练习层（/{subject}/questions）

学科指南页 ── 独立（/{subject}/guide）
```

### 3.2 页面 1：思维方法卡片页

**路由**：`/{subject}/methods/{id}`
**数据来源**：内容模板第四章 JSON

**页面区块**（6个一级区块）：

| # | 区块名 | 层次 | 数据字段 |
|:---:|:---|:---|:---|
| 1 | 一句话定义 | 顶部摘要 | `definition` |
| 2 | 学科本质 | 认知层 | `essence`（~200字） |
| 3 | 触发条件 | 应用层 | `triggers[]`（列表） |
| 4 | 训练阶梯 | 练习层 | `trainingLadder.basic/intermediate/challenge` |
| 5 | 跨板块映射 | 迁移层 | `crossChapterMapping[]`（表格） |
| 6 | 自我检测 | 自评层 | `selfCheck.q1_concept/q2_transfer/q3_diagnosis` |

**底部链接**：
- `relatedModels[]` → 跳转到关联模型详解页
- `relatedParadigms[]` → 跳转到关联分析范式页

### 3.3 页面 2：知识节点详解页

**路由**：`/{subject}/concepts/{id}`
**数据来源**：内容模板第五章 JSON

**页面区块**（5个模块，前端渲染为5个一级区块）：

| # | 区块名 | 层次 | 数据字段 |
|:---:|:---|:---|:---|
| 1 | 前置知识检测 | 入口层 | `preCheck[]`（3道快速检测题） |
| 2 | 叙事正文 | 认知层（核心） | 6个字段，见下方 |
| 3 | 分层变形 | 练习层 | `variations[]`（B/J/T三级） |
| 4 | 公式卡片 | 工具层 | `formulas[]`（嵌入式，不独立为页面） |
| 5 | 理解度自评 | 自评层 | `selfEval[]`（A/B/C三级） |

**叙事正文的6个段落**（前端以标题独立呈现，不显示编号）：

| 段落 | 前端标题 | 数据字段 | 字数 |
|:---:|:---|:---|:---|
| 1 | 情境锚定 | `context` | 100-150 |
| 2 | 困惑预设 | `confusion` | 150-200 |
| 3 | 实验/现象呈现 | `experiment` | 200-250 |
| 4 | 概念涌现 | `concept` | 150-200 |
| 5 | 推导叙事 | `derivation` | 300-400 |
| 6 | 迁移与应用 | `transfer` | 150-200 |

**数学公式渲染**：所有公式使用 KaTeX，变量首次出现需标注含义。

**关联链接**：
- `relatedModels[]` → 跳转到关联模型详解页
- `crossLinks[]` → 跨学科关联（如 K08 ↔ 数学·向量运算）

### 3.4 页面 3：模型详解页（最复杂）

**路由**：`/{subject}/models/{id}`
**数据来源**：内容模板第六章 JSON

**页面区块**（内容方生产7个模块，前端渲染为**6个一级区块**）：

> **关键**：模块3「分层变形」嵌套在模块2「原理讲解」内部展示。内容方提交7个模块的完整JSON，前端负责嵌套渲染。

| # | 前端区块 | 内容模块 | 层次 | 子项 |
|:---:|:---|:---:|:---|:---|
| 1 | 模型卡片 | 模块1 | 工具层 | 定位/核心定义/工具箱/识别关键词/典型陷阱/变式索引 |
| 2 | 原理讲解 | 模块2 | 认知层 | 六段式叙事正文 + 前置知识检测 + 分层变形 + 理解度自评 |
| 3 | 知识网络 | 模块4 | 认知层 | 前置→核心→后续关系图 + 公式关系图（纯可视化） |
| 4 | 方法论 | 模块5 | 方法层 | 探索引例/审题清单/解题流程图/公式决策表/核心方法论/口诀 |
| 5 | 自我检测 | 模块6 | 检测层 | 3道费曼式检测题（概念复述/即时应用/理解确认） |
| 6 | 生活应用 | 模块7 | 拓展层 | 科技前沿 + 日常生活 + 安全防护（至少2个案例） |

**区块1 模型卡片 详细结构**：

```
📍 模型定位
    ├── 所属板块（学期·板块名）
    ├── 前置知识：[依赖的知识节点编号] → 可点击跳转知识节点页
    └── 后续关联：[会被哪些模型用到] → 可点击跳转模型页

🎯 核心定义
    └── 一句话精确定义 + 适用条件列表

🛠️ 工具箱（⭐ 本模型唯一公式汇总处）
    └── 表格：公式形式 | 使用场景 | 公式内容

🔑 识别关键词
    └── 列表：关键词 → 物理含义

⚠️ 典型陷阱提示
    └── 列表：陷阱名称 → 一句话说明 + 正确做法

📋 变式索引
    └── B级：名称 / J级：名称 / T级：名称
```

**区块2 原理讲解 内部嵌套结构**：

```
原理讲解
├── 【前置知识检测】3道快速检测题
├── 【六段式叙事正文】（同知识节点，标题独立展示）
│   ├── 情境锚定 → 困惑预设 → 实验/现象呈现
│   ├── 概念涌现 → 推导叙事 → 迁移应用
├── 【分层变形】⭐ 三层完整实例
│   ├── B级变形：一步推导 + 2道例题
│   ├── J级变形：多步推理 + 2道例题
│   └── T级变形：综合分析 + 2道例题
└── 【理解度自评】⭐ A/B/C 三级
```

**区块5 方法论 详细结构**：

```
方法论（⭐ 思维显性化的关键）
├── 1. 探索引例：1个对比题（启发性）
├── 2. 审题要点清单：5步检查（每步标注关联思维方法）
├── 3. 解题流程图：标准流程
├── 4. 公式选择决策表：按已知条件查表（含"思维导航"列）
├── 5. 核心方法论：2-3条（每条标注学科核心思维方法）
└── 6. 方法速查口诀：5条以内
```

**侧边栏入口**：
- 模型详解页提供侧边栏入口「分层题库」，点击跳转到题库页并自动筛选本模型的题目
- 题库页结构：B级基础8题 + J级进阶5题 + T级挑战3题

### 3.5 页面 4：分析范式页

**路由**：`/{subject}/paradigms/{id}`
**数据来源**：内容模板第七章 JSON

**呈现方式**：双重呈现
1. **独立页面**（本路由）：完整展示五段内容
2. **题库解析内嵌组件**：学生在做题时点击"解法提示"，以摘要卡片展示核心内容，提供"查看完整范式"链接

**页面区块**（5个一级区块）：

| # | 区块名 | 数据字段 | 说明 |
|:---:|:---|:---|:---|
| 1 | 触发信号 | `trigger` | "看到什么关键词→想到这个范式" |
| 2 | 思考路径 | `path[]` | 3-7步标准化步骤，每步标注 `thinkingMethod` |
| 3 | 变式预警 | `variationWarning` | 条件怎么变→思路怎么调 |
| 4 | 错因图谱 | `errorMap[]` | 表格：典型错误思路 / 认知根源 / 正确路径 |
| 5 | 本质回溯 | `essence` | 范式的底层原理 |

**顶部元数据**：
- `name`：范式名称
- `model`：所属模型 → 可跳转到模型详解页
- `thinkingMethod`：关联思维方法 → 可跳转到思维方法卡片页
- `level`：适用难度层级（B/J/T）

### 3.6 页面 5：题库页

**路由**：`/{subject}/questions`
**数据来源**：题库总体架构v1.2 JSON

#### 五维筛选体系

| 维度 | 代码 | 筛选方式 |
|:---|:---|:---|
| 能力层级 | level | L1 诊断 / L2 模型题 / L3 综合 |
| 学习目标 | target | 同步SYNC / 期末EXAM / 高考GAOKAO / 强基FOUNDATION / 竞赛COMPETE |
| 题目功能 | function | 诊断DIAG / 巩固PRACTICE / 变式VARIATION / 综合INTEGRATED / 真题REAL / 方法METHOD |
| 难度 | difficulty | D1(识记) ~ D6(压轴)，滑块或下拉 |
| 题型 | type | 按学科展示对应题型选项 |

#### 学科代码

| 代码 | 学科 |
|:---|:---|
| PHY | 物理 |
| MAT | 数学 |
| CHE | 化学 |
| BIO | 生物 |
| CHN | 语文 |
| ENG | 英语 |

#### 题目ID格式

```
{学科代码}-{模型编号}-{难度层级}-{题型代码}-{序号}
示例：PHY-M01-B-SC-001（物理·M01·B级·单选题·第1题）
```

> **数学特殊**：模型编号位置使用问题类型编码，如 `MAT-F3-B-CL-001`

#### 题目展示

每道题的JSON核心字段：

| 字段 | 类型 | 说明 |
|:---|:---|:---|
| `id` | string | 唯一标识 |
| `content` | string | 题干（支持KaTeX） |
| `type` | enum | 题型代码（见下方题型枚举） |
| `options` | array | 选项（选择题用） |
| `answer` | string | 标准答案 |
| `explanation` | string | 解析（五段式或四段式） |
| `hint` | string | 提示（按层级控制展示时机） |
| `difficulty` | 1-6 | 难度数值 |
| `level` | L1/L2/L3 | 能力层级 |
| `tags` | string[] | 标签（知识点、方法、错误类型等） |
| `timeLimit` | number | 建议用时（秒） |

## 四、知识网络链接体系

### 4.1 六类双向链接

页面之间的导航不是硬编码的，而是通过数据中的**六类双向链接**自动生成：

| 链接类型 | 源 → 目标 | JSON字段 | 示例 |
|:---|:---|:---|:---|
| ① 节点→模型 | 知识节点 → 关联模型 | `relatedModels` | K09 → M05/M06/M07/M08 |
| ② 模型→节点 | 模型 → 前置知识节点 | `prerequisites` | M05 → [K08, K09] |
| ③ 模型→范式 | 模型 → 关联范式 | 知识网络映射表 | M05 → [R09, R10, R11] |

### 4.3 三条自然发现路径

网站应支持学生从不同入口自然发现内容：

**路径一：从概念出发**
知识节点 → "这个概念用在哪些场景？" → 关联模型 → "具体怎么分析？" → 关联范式 → "底层思维是什么？" → 思维方法卡片

**路径二：从问题出发**
题库做题 → 卡住 → 点击"解法提示" → 分析范式摘要卡片 → "深入学习思维方法" → 思维方法卡片

**路径三：从思维方式出发**
思维方法卡片 → "用到这个思维的模型有哪些？" → 模型列表 → 学生选择薄弱板块进入

## 六、三层能力模型（B/J/T 分级）

### 6.1 能力层级定义

| 层级 | 名称 | 行为特征 | 题目特征 | 占比 |
|:---|:---|:---|:---|:---:|
| L1 / B级 | 知识掌握 | 识别知识点 + 套用公式 | 标准题型、单一知识点 | 40% |
| L2 / J级 | 思维运用 | 识别模型 + 选择方法 + 多步推理 | 综合题、多知识点组合 | 45% |
| L3 / T级 | 迁移创新 | 陌生情境中识别底层结构 | 创新题、情境题、开放题 | 15% |

### 6.2 诊断标准

- L1：正确率 ≥ 90%，5分钟内完成基础题
- L2：正确率 ≥ 60%，能识别模型并完成2-3步推理
- L3：正确率 ≥ 40%，能在全新情境中识别底层知识结构

### 6.3 六级难度与B/J/T映射

| 等级 | 代码 | 说明 | 时间参考 | 对应B/J/T |
|:---:|:---|:---|:---|:---:|
| 1 | D1 | 概念识记，直接套公式 | 30秒 | — |
| 2 | D2 | 基本理解，简单辨析 | 45秒 | B级 |
| 3 | D3 | 模型套用，两步推导 | 60秒 | B级 |
| 4 | D4 | 多步综合，跨知识点 | 90秒 | J级 |
| 5 | D5 | 高考难题，多模型组合 | 120秒 | T级 |
| 6 | D6 | 压轴/强基/竞赛 | 180秒 | — |

> **语/英时间特殊**：英语读后续写25分钟、概要15分钟、作文20分钟；语文作文60分钟、鉴赏15-20分钟。这些T级输出题不适用秒级参考。
