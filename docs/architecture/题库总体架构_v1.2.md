# 星耀高中教育平台 — 题库总体架构 v1.2

> 定位：覆盖六科全阶段、全目标、全题型的完整题库体系设计
> 与《物理题库设计规范_v3.1（新课L2+L3）》的关系：v3.1是本文的一个子集（物理·L2+L3+L1+EXAM层）
> 与各科知识网络的关系：物理56节点/43模型、化学57节点/48模型、生物52节点/48模型、数学~75节点/58问题类型、英语49节点/52模型、语文43节点/29模型
> 设计原则：分层递进、六科统一、目标导向、与网站功能深度对接、学科特性差异化处理

## 一、题库的定位

题库不是知识内容的附属品。学生来到星耀，最终要做的是**做题→理解→进步**。题库是学习系统的**核心交付物**和**效果检验手段**。

### 题库与网站模块的对接关系

```
┌─────────────────────────────────────────────────────────────┐
│                      星耀学习闭环                             │
│                                                             │
│  知识详解(模型页) ──→ 认知图谱 ──→ 解题套路 ──→ 题库练习     │
│       ↑                                          │          │
│       └──────────── 错题本 ←── 学习报告 ←───────┘          │
│                                                             │
│  高考专项 ──→ 强基计划 ──→ 竞赛专区                          │
└─────────────────────────────────────────────────────────────┘
```

## 二、五维分类体系

每一道题同时具有五个维度的标签，形成多维检索体系：

### 维度一：学科（Subject）

| 学科代码 | 学科 | 特殊题型说明 |
|:---|:---|:---|
| PHY | 物理 | 计算题占比较大、实验题、多过程综合题、图像分析题 |
| CHE | 化学 | 推断题、方程式书写、实验流程设计、工业流程分析、有机推断与合成 |
| MAT | 数学 | 证明题、构造题、多解法题、新定义题、开放性试题、尺规作图 |
| BIO | 生物 | 图表分析题、实验设计题、遗传推断题、遗传系谱分析、材料分析 |
| CHN | 语文 | 阅读理解、古诗文默写与鉴赏、作文、文言文阅读、语言文字运用 |
| ENG | 英语 | 完形填空、语法填空、阅读理解、七选五、读后续写、概要写作、听力、翻译 |

### 维度二：能力层级（Level）

与星耀三层能力模型严格对应：

| 层级 | 代码 | 说明 | 对应网站 | 对应v3.0 |
|:---|:---|:---|:---|:---|
| **L1 知识掌握** | L1 | 概念辨析、公式记忆、基础理解 | 知识详解页·自我检测 | ❌ v3.0未覆盖 |
| **L2 思维运用** | L2 | 模型识别、套路套用、综合分析 | 练习中心·B/J/T | ✅ v3.0全部覆盖 |
| **L3 迁移创新** | L3 | 跨模型组合、开放题、压轴突破 | 练习中心·T层(扩展) | ⚠️ v3.0部分覆盖 |

### 维度三：学习目标（Target）

| 目标 | 代码 | 说明 | 对应网站模块 |
|:---|:---|:---|:---|
| 同步学习 | SYNC | 跟课堂进度，课后巩固 | 练习中心·知识点练习、章节测试 |
| 期末考试 | EXAM | 阶段性检测，模拟考试 | 练习中心·章节测试 |
| 高考备考 | GAOKAO | 高考真题训练、应试技巧 | 高考专区(规划中)、练习中心·考试真题 |
| 强基校测 | FOUNDATION | 校测风格、超纲拓展 | 强基计划专区(规划中) |
| 学科竞赛 | COMPETE | 预赛/复赛/决赛 | 竞赛专区(规划中) |

### 维度四：题目功能（Function）

| 功能 | 说明 | 出题要求 |
|:---|:---|:---|
| **诊断题** | 快速检测知识掌握度 | 2-3题一组，覆盖核心概念，即做即判 |
| **巩固题** | 重复训练形成熟练度 | 同一模型、同一套路，变式重复 |
| **变式题** | 换条件/场景，训练迁移能力 | 改变初始条件、物理情境、约束条件 |
| **综合题** | 多知识混合，考串联能力 | 跨模型、跨章节、跨方法 |
| **真题** | 真实考试题，训练手感 | 标注来源年份、保留原题风格 |
| **方法题** | 一题多解、多题一法 | 展示不同解法思路，训练方法选择 |

### 维度五：难度分级（Difficulty）

采用统一的六级难度体系，**允许学科差异校准**：

| 等级 | 代码 | 说明 | 时间参考 | 对应v3.0 |
|:---|:---|:---|:---|:---|
| 1 | D1 | 概念识记，直接套公式 | 30秒/题 | — |
| 2 | D2 | 基本理解，简单辨析 | 45秒/题 | B层 |
| 3 | D3 | 模型套用，两步推导 | 60秒/题 | B层 |
| 4 | D4 | 多步综合，跨知识点 | 90秒/题 | J层 |
| 5 | D5 | 高考难题，多模型组合 | 120秒/题 | T层 |
| 6 | D6 | 压轴突破/强基/竞赛 | 180秒/题 | v3.0未覆盖 |

> **与v3.0的映射关系**：B层=D2+D3，J层=D4，T层=D5。D1(概念题)和D6(超难题)是新增层。

#### 学科差异校准（T级映射扩展）

| 学科 | T级映射 | 特殊时间参考 | 说明 |
|:---|:---:|:---|:---|
| **物理/化学/数学** | T=D5 | 120-180秒/题 | 保持原映射 |
| **生物** | T=D5~D6 | 120-180秒/题（实验设计/材料分析） | 实验设计题复杂度可能超D5 |
| **英语** | T=D5~D6 | **输出题：15-30分钟/题**；阅读题：120-180秒/题 | 读后续写/概要/作文时间远超常规 |
| **语文** | T=D5~D6 | **作文：50-60分钟/题**；鉴赏：15-20分钟/题 | 作文60分/60分钟，古诗文鉴赏主观性强 |

> 英语T级输出题（读后续写25分/25分钟，概要10分/15分钟，作文15分/20分钟）和语文T级作文（60分/60分钟）的时间参考需单独定义，不适用于常规秒级参考。

## 五、JSON数据模型（统一结构）

### 5.1 题目核心字段

```typescript
interface Question {
  // ====== 标识 ======
  id: string              // 如 'PHY-M01-B-SC-001'（学科-模型-层级-题型-序号）
  subject: SubjectCode   // PHY/CHE/MAT/BIO/CHN/ENG

  // ====== 归属 ======
  modelId?: string        // 所属模型，如 'PHY-M01'
  knowledgeIds: string[]  // 关联知识点
  strategyId?: string     // 关联解题套路
  chapterId?: string      // 所属章节

  // ====== 五维标签 ======
  level: 'L1' | 'L2' | 'L3'
  target: 'SYNC' | 'EXAM' | 'GAOKAO' | 'FOUNDATION' | 'COMPETE'
  function: 'DIAG' | 'PRACTICE' | 'VARIATION' | 'INTEGRATED' | 'REAL' | 'METHOD'
  difficulty: 1 | 2 | 3 | 4 | 5 | 6

  // ====== 题目内容 ======
  type: QuestionType      // 见5.2节扩展枚举
  content: string        // 题干（支持KaTeX数学公式）
  options?: QuestionOption[]  // 选项（选择题用）
  answer: string          // 标准答案
  explanation: string     // 解析（五段式：考什么→怎么做→为什么→避坑→答案）

  // ====== 学科扩展字段 ======
  // --- 英语/语文 语言学科 ---
  audioUrl?: string;        // 听力题音频（英语）
  wordCount?: number;       // 写作题字数要求（语/英）
  rubric?: string[];        // 主观题评分维度（语/英）
  sourceText?: string;      // 读后续写/概要/文言文的前文
  genre?: string;           // 语篇类型（narrative/expository/argumentative）

  // --- 化学/生物/物理 实验学科 ---
  apparatus?: string[];     // 实验器材清单（化学/生物/物理实验题）
  procedure?: string[];     // 实验步骤（用于排序/设计题）
  diagramUrl?: string;      // 装置图/曲线图/系谱图（物/化/生）

  // --- 数学 证明/作图 ---
  proofSteps?: string[];    // 证明题分步评分点
  canvasRequired?: boolean; // 是否需要Canvas作图（几何/函数）

  // --- 全学科 方法题 ---
  alternativeSolutions?: string[]; // 一题多解的备选解法
  methodTags?: string[];    // 方法标签（换元/数形结合/分类讨论等）

  // ====== 辅助 ======
  hint?: string           // 提示（做不出来时看）
  timeLimit?: number      // 建议用时（秒），T级输出题单独定义分钟级
  source?: string         // 来源（真题标注年份和卷别）
  tags: string[]          // 标签（知识点、方法、错误类型等）
  points?: string[]       // 考查要点（分步给分用）
}
```

### 5.2 题型枚举扩展（QuestionType）

```typescript
type QuestionType = 
  // 通用题型
  | 'single'           // 单选题
  | 'multiple'         // 多选题
  | 'fill'             // 填空题
  | 'calculation'      // 计算题
  | 'proof'            // 证明题
  | 'essay'            // 论述/作文题（通用）

  // 物理特有
  | 'image_analysis'   // 图像分析题（v-t图、F-x图、电场线图）
  | 'multi_process'    // 多过程综合计算题
  | 'experiment_design'// 实验设计题（自选器材、设计步骤）
  | 'optional'         // 选做题（3-3热学/3-4光学振动）

  // 化学特有
```

## 七、内容生产策略

### 7.1 生产顺序

```
第一步：物理打样（已确定）
  └── v3.0规范的L2模型题 → 交给外部AI生产

第二步：物理完整化
  └── 补充L1诊断题 + L3综合题 → 确立物理完整样本

第三步：化学/生物同步跟进
  └── 参照物理样本，快速复制到化学和生物

第四步：数学/英语/语文
  └── 语言学科题型差异大，需要单独设计样本
  └── 数学无传统模型层，以58个问题类型为骨架组织题目
  └── 英语基于v1.3知识网络（49节点/52模型/53范式）
  └── 语文基于v1.1知识网络（43节点/29文体模型/45分析范式）

第五步：高考专项
  └── 真题收集为主，辅以模拟题

第六步：强基+竞赛（长期）
  └── 高价值、低频需求，持续积累
```

### 7.2 六科样本设计要点

| 学科 | 样本设计核心差异 | 生产难点 | 特殊字段需求 |
|:---|:---|:---|:---|
| **物理** | 条件充分性、单位统一、图像规范 | 公式渲染（LaTeX）、受力分析图 | diagramUrl（图像题） |
| **化学** | 方程式配平判分、实验装置图、工业流程图 | 化学式/方程式标准化输入、流程图交互 | equation（方程式书写）、apparatus（器材清单）、diagramUrl |
| **生物** | 图表分析（曲线/柱状/表格）、实验设计对照原则 | 遗传系谱图绘制、显微图像识别 | diagramUrl（系谱图/显微图）、procedure（实验步骤） |
| **数学** | 多选题部分得分、证明题步骤分、新定义题现场学习 | 公式推导过程展示、几何作图交互 | proofSteps（证明分步）、canvasRequired（Canvas作图） |
| **英语** | 语篇片段为题干、干扰项设计依赖中式陷阱、写作题多维评分 | 听力音频管理、写作自动评分（或人工评） | audioUrl（听力音频）、wordCount/rubric/sourceText/genre（写作题） |
| **语文** | 主观题占比极高、古诗文鉴赏情感分析、作文评分 | AI评分准确性、文言文断句交互 | rubric（鉴赏分维度评分）、sourceText（文言文原文） |

### 7.3 质量标准

每道题必须通过以下检查：

1. **准确性**：答案和解析100%正确
2. **严谨性**：条件充分、无歧义、学科表达规范
3. **层级匹配**：难度与标注一致（注意语/英/生T级可映射D6）
4. **解析质量**：不只给答案，更要讲清楚"为什么这样做"
5. **标签完整**：五维标签齐全，学科扩展字段按需填充
6. **来源标注**：真题必须标注年份和卷别，英语/语文例句必须标注出处

### 7.4 用户决定

- **物理L2模型题**：另找AI生产（已确定），v3.0规范是执行手册
- **其他所有题目**：是否也在外部生产，还是后续再定
