# 物理模型内容模板

> 适用于星耀平台各科目内容建设，物理科目已用此模板验证（v3.0）

---

## 一、模板结构

每个「知识元」按六段式展开：

| 段 | 名称 | 目的 | 典型篇幅 |
|----|------|------|----------|
| 1 | 引入 | 从生活/实验现象切入，激发兴趣 | 1-2段 |
| 2 | 是什么 | 核心概念定义，简洁精准 | 1段 |
| 3 | 怎么用 | 适用场景 + 公式 + 注意事项 | 2-3段 |
| 4 | 公式 | 独立模块，含推导过程 | 公式卡片式 |
| 5 | 纵横联系 | 与其他知识元的关联 | 1-2段 |
| 6 | 检测 | 费曼学习法引导（非刷题） | 3-5个问题 |

---

## 二、字段规范

### 基础字段

```typescript
{
  id: string          // 如 "M01"
  name: string         // 模型名称，如 "传送带模型"
  alias?: string[]     // 别名，如 ["传送带", "传送带传动"]
  chapter: string      // 所属章节，如 "直线运动"
  section?: string     // 所属小节（可选）
  difficulty: number   // 难度 1-6

  // 六段式内容
  intro: string        // 引入：现象/故事
  concept: string      // 是什么：定义
  application: string  // 怎么用：场景/步骤
  formulas: Formula[] // 公式卡片
  relations: string    // 纵横联系

  // 检测（费曼学习法）
  fermiQuestions: string[]  // 3-5个引导问题

  // 元数据
  tags: string[]       // 搜索标签
  related: string[]    // 关联模型 ID，如 ["M02", "M05"]
  media?: string       // 配图/视频 URL
}
```

### 公式卡片字段

```typescript
{
  formula: string      // LaTeX 公式，如 "v = v_0 + at"
  name: string         // 公式名称，如 "速度-时间关系"
 适用条件: string      // 如 "匀变速直线运动"
  derivation?: string  // 推导过程（可选）
  variants?: string[]  // 常见变形
  unit?: string        // 各物理量单位
}
```

## 三、示例（以「传送带模型」为例）

### 基础信息

- **编号**：M08
- **名称**：传送带模型
- **别名**：传送带传动、水平传送带
- **章节**：力学 — 相互作用与运动
- **难度**：⭐⭐⭐⭐（D4）

### 六段式内容

**① 引入**
> 工厂流水线上，产品随传送带匀速运动；机场行李转盘上，行李随转盘一起运动。传送带上的物体，运动状态取决于什么？

**② 是什么**
> 传送带模型指：物体在传送带上运动时，由于传送带提供摩擦力，物体可能与传送带保持相对静止（一起运动）或发生相对滑动的情景。

**③ 怎么用**
> 解题步骤：
> 1. 分析传送带运动方向和速度
> 2. 判断物体初速度与传送带速度的大小关系
> 3. 计算临界摩擦力 f = μN，比较最大静摩擦与所需摩擦力
> 4. 若 f_max ≥ 所需摩擦 → 一起匀速/加速；否则 → 相对滑动
> 5. 运动过程分段处理，注意速度相等的转折点

> ⚠️ 常见错误：忽略速度相等后的运动状态变化（从滑动变为一起运动）

**④ 公式**

| 公式 | 名称 | 适用条件 |
|------|------|----------|
| a = μg | 加速度 | 滑动阶段，传送带与物体速度不同 |
| v = at + v₀ | 速度公式 | 匀变速阶段 |
| s = ½at² + v₀t | 位移公式 | 匀变速阶段 |

推导过程：
- 对物体受力分析：重力 G、弹力 N、摩擦力 f
- 竖直方向 N = mg
- 水平方向 f = ma，滑动时 f = μN = μmg
- 得 a = μg

**⑤ 纵横联系**

- **向上联系**（本章）：→ 「板块模型 M09」（无传送带牵引力，板块间摩擦驱动）
- **向下延伸**（本章）：→ 「连接体问题 M15」（多物体共享加速度）
- **跨学科**：→ 「传送带功率问题」（功能关系，与功率公式 P = Fv 结合）

**⑥ 检测（费曼学习法）**

> 不查资料、不看笔记，用自己的话回答：
>
> 1. 传送带上的物体，什么情况下会一直滑下去，什么情况下会最终和传送带一起运动？
> 2. 「传送带突然加速」和「物体突然获得初速度」，结果一样吗？为什么？
> 3. 如果传送带光滑，物体还能被带动吗？这时物体怎么运动？
> 4. 传送带模型的本质是什么？它和「板块模型」的核心区别在哪？

## 四、内容质量标准

### 引入段
- ✅ 从生活/实验现象出发，有画面感
- ✅ 能引发思考，不是直接讲定义
- ✅ 1-2段足够，不冗长

### 概念段
- ✅ 严格但不拗口
- ✅ 关键条件不遗漏
- ❌ 避免"顾名思义"——学生常见错误

### 怎么用段
- ✅ 步骤化，1234排列
- ✅ 包含常见错误提示（⚠️）
- ✅ 有判断标准（如"比较最大静摩擦与所需摩擦力"）

### 公式段
- ✅ 主公式突出
- ✅ 适用条件明确标注
- ✅ 推导过程关键步不跳过
- ✅ 常见变形列出

### 纵横联系段
- ✅ 向上（基础模型）、向下（进阶模型）、横向（同级别模型）都要有
- ✅ 跨学科联系加分

### 检测段
- ❌ 不出计算题
- ❌ 不出选择填空
- ✅ 问题触发"能不能讲清楚"
- ✅ 问题层次：从记忆→理解→迁移

## 六、迁移说明

此模板同样适用于化学、生物科目：

**化学科目调整**：
- 「引入」→ 实验现象或工业生产场景
- 「公式」→ 化学方程式、平衡常数等
- 「纵横联系」→ 跨反应类型、跨物质类别

**生物科目调整**：
- 「引入」→ 自然现象或生活案例
- 「公式」→ 数量关系式、概率计算等
- 「检测」→ 概念复述 + 图示描述

**模板复用优先级**：物理 → 化学 → 生物 → 数学 → 英语/语文
