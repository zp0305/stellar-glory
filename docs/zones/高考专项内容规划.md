# 星耀高中教育平台 — 高考专项内容规划

> 定位：为"高三专题"模块和"练习中心·考试真题"模块提供内容蓝图
> 适用范围：六科通用（本文件以物理为例，其他五科复制相同结构）
> 与《网站内容框架v2.0》的关系：本文是v2.0中"高三专题"和"考试真题"的详细展开

## 一、高考专项内容总览

### 为什么需要高考专项

星耀的核心是"理解型学习"（知识网络→模型→套路），帮助学生建立物理图景。但高考不仅考理解，还考：
- **政策信息**：新高考怎么考、赋分怎么算、选科怎么选
- **应试能力**：时间分配、答题规范、步骤得分策略
- **真题实战**：真题手感不同于日常练习，需要专门训练
- **升学规划**：目标院校、强基计划、专业选科要求

这些内容不属于"学科知识"，但直接影响高考成绩和升学结果。

### 高考专项 vs 学科内容的分工

```
学科内容（已有/在建）
├── 认知图谱：建立知识地图
├── 模型详解：深入理解每个模型
├── 解题套路：掌握解题方法
├── 题库练习：日常训练
└── 错题本：查漏补缺

高考专项（本文规划）
├── 高考政策与命题趋势：看清方向
├── 高考专题突破：题型专项训练
├── 应试策略与答题模板：考场技巧
├── 模拟考试：实战演练
└── 升学路径与院校参考：定目标
```

## 二、五大内容类别详细规划

### 类别1：高考政策与命题趋势

#### 放置位置
- **高三专题页**（`/physics/senior`）的子模块之一
- 数据文件：`src/data/physics/senior.ts` 中的 `policyTrends` 数组

#### 内容条目

| 编号 | 条目 | 内容要点 | 更新频率 |
|:---|:---|:---|:---|
| P01 | 新高考改革政策解读 | 3+1+2模式、选科要求、赋分制原理、合格考vs等级考 | 每年更新 |
| P02 | 各省高考模式一览 | 哪些省用全国卷（甲/乙/新课标I/II）、哪些省自主命题、物理在选科中的地位 | 每年更新 |
| P03 | 物理学科考纲与考查要求 | 必修+选择性必修的考查范围、实验考查清单、必考vs选考内容 | 每年更新 |
| P04 | 近五年命题趋势分析 | 情境化命题趋势、反套路考法、跨学科融合倾向、实验考查变化、计算量变化 | 每年更新 |
| P05 | 新题型与考法变化 | 近两年出现的新题型（如：开放性试题、评价性试题、多过程综合题） | 每年更新 |

#### 每条内容的写作标准
- **篇幅**：800-1500字
- **风格**：信息密集、条理清晰，用表格对比代替长段文字
- **禁止**：不含主观评论，只做客观分析和数据呈现
- **来源标注**：每条内容必须标注信息来源（教育部文件、考试院公告等）

#### 数据结构

```typescript
interface PolicyTrend {
  id: string;                    // 如：P01
  title: string;                 // 条目标题
  category: 'policy' | 'exam_analysis' | 'trend' | 'new_type';
  subject: string;               // 学科
  content: string;               // 正文（Markdown格式）
  source: string;                // 信息来源
  updateYear: number;            // 更新年份
  tags: string[];                // 标签
}
```

### 类别2：高考专题突破

#### 放置位置
- **高三专题页**（`/physics/senior`）的子模块之一
- 同时与**练习中心·考试真题**联动（每个专题可一键跳转到对应训练）
- 数据文件：`src/data/physics/senior.ts` 中的 `examTopics` 数组

#### 内容条目

| 编号 | 专题 | 覆盖范围 | 预估题量 |
|:---|:---|:---|:---|
| T01 | 选择题速解技巧 | 多选题的排除法、极值法、量纲分析法、图像法、特殊值法 | 30道 |
| T02 | 实验题专项 | 力学实验（验证牛顿定律、测量动摩擦因数等）、电学实验（伏安法、测电源电动势等）、光学实验 | 25道 |
| T03 | 力学计算大题 | 运动学综合、牛顿定律综合、能量动量综合、多过程问题 | 20道 |
| T04 | 电磁学计算大题 | 带电粒子运动、电磁感应综合、交流电路 | 20道 |
| T05 | 选做题专项（热学） | 理想气体状态方程、热力学定律应用 | 15道 |
| T06 | 选做题专项（振动与波/光学） | 机械波传播、光的折射与全反射、光的干涉与衍射 | 15道 |
| T07 | 基础题保分训练 | 必拿分的简单题（概念辨析、公式直接应用、单位换算） | 30道 |
| T08 | 中档题提分训练 | 需要综合2-3个知识点的中档题 | 30道 |
| T09 | 压轴题突破 | 多过程+多模型组合的高难度题 | 15道 |

#### 每个专题的内部结构

```
专题页（如 T03-力学计算大题）
├── 专题概述
│   → 考频统计（近5年高考中出现的频率和分值分布）
│   → 题型特征（这类题长什么样、一般怎么问）
│   → 能力要求（需要掌握哪些知识和模型）
│
├── 解题策略
│   → 通用解题流程（读题→画图→列式→求解→检验）
│   → 关键技巧（每个专题特有的技巧，如：力学大题的受力分析顺序）
│   → 时间分配建议（这类题建议花多少分钟）
│
├── 典型真题精讲（3-5道）
│   → 每道题包含：题目+解题过程+评分标准+常见失分点
│
├── 限时训练（10-15道）
│   → 按难度梯度排列
│   → 可一键跳转到练习中心对应题目
│
└── 薄弱诊断
    → 做完后给出薄弱知识点和推荐复习模型
```

#### 写作标准
- **篇幅**：每个专题2000-3500字（不含题目）
- **真题精讲**：每题包含完整解题过程，标注"可得分步骤"和"易失分点"
- **时限感**：所有训练标注建议用时，培养时间管理意识

#### 数据结构

```typescript
interface ExamTopic {
  id: string;                    // 如：T01
  title: string;                 // 专题名称
  subtitle: string;              // 一句话描述
  subject: string;               // 学科
  topicType: 'choice' | 'experiment' | 'calculation' | 'optional' | 'level_based';
  
  // 专题内容
  overview: string;              // 专题概述（考频、题型特征、能力要求）
  strategy: string;              // 解题策略（流程+技巧+时间建议）
  
  // 典型真题
  modelQuestions: {
    questionId: string;          // 关联题库中的题目ID
    analysis: string;            // 详细解题过程
    scoringPoints: string[];     // 得分步骤
    commonMistakes: string[];    // 常见失分点
  }[];
  
  // 训练题
  practiceQuestionIds: string[]; // 关联题库中的训练题ID
  suggestedTime: number;         // 建议总用时（分钟）
  
  // 元数据
  frequency: number;             // 近5年考频（出现次数）
  avgScore: number;              // 平均得分率（%）
  relatedModelIds: string[];     // 关联的模型ID
  tags: string[];
}
```

### 类别3：应试策略与答题模板

#### 放置位置
- **高三专题页**（`/physics/senior`）的子模块之一
- 数据文件：`src/data/physics/senior.ts` 中的 `examStrategies` 数组

#### 内容条目

| 编号 | 条目 | 内容要点 |
|:---|:---|:---|
| S01 | 物理考场时间分配方案 | 选择题（约25分钟）+ 实验题（约10分钟）+ 计算题（约35分钟）+ 检查（约5分钟）的分配策略，根据难度灵活调整 |
| S02 | 选择题答题技巧 | 多选题的保守策略（不确定的不选）、排除法流程、量纲分析快速排除、图像题读图技巧 |
| S03 | 实验题答题模板 | 实验目的写法、仪器读数规范、误差分析标准格式、数据处理（描点作图/公式法）的步骤化模板 |
| S04 | 计算大题答题模板 | "已知-求-解-答"四段式写法、公式引用规范（原始公式vs推导公式）、单位换算规范、有效数字处理 |
| S05 | 分步得分策略 | 哪些步骤写了一定有分、哪些步骤没写一定没分、中间结果算错但公式正确能否得分、"跳步"的风险 |
| S06 | 常见失分点与避坑指南 | 物理中高频失分的20个坑（如：g取值不统一、单位漏写、矢量方向忘标等） |
| S07 | 考前一周冲刺建议 | 最后7天每天做什么、哪些内容值得再看、哪些内容不要碰、心态调整 |

#### 每条内容的写作标准
- **篇幅**：600-1200字
- **风格**：极度实用，直接给"怎么做"，不给"为什么"
- **模板化**：能用模板/流程图表达的坚决不用长段落
- **案例**：每条至少包含1-2个具体案例（正例+反例对比）

#### 数据结构

```typescript
interface ExamStrategy {
  id: string;                    // 如：S01
  title: string;
  subtitle: string;
  subject: string;
  strategyType: 'time_management' | 'technique' | 'template' | 'scoring' | 'pitfall' | 'review';
  content: string;               // 正文（Markdown，可含表格、流程图描述）
  templates?: string[];          // 答题模板（如果有）
  doAndDont?: {
    do: string[];                // 推荐做法
    dont: string[];              // 避免做法
  };
  examples?: {
    good: string;                // 正例
    bad: string;                 // 反例
    note: string;                // 点评
  }[];
  tags: string[];
}
```

### 类别4：模拟考试

#### 放置位置
- **练习中心·考试真题**模式（`/physics/practice/quiz?mode=exam`）中新增"模拟考试"子类型
- 数据文件：`src/data/physics/mockExams.ts`

#### 内容规划

| 编号 | 试卷名称 | 试卷结构 | 难度定位 | 预估题量 |
|:---|:---|:---|:---|:---|
| ME01 | 模拟卷·基础达标 | 完整高考结构，难度偏简单 | 巩固基础 | 15题 |
| ME02 | 模拟卷·中等水平 | 完整高考结构，难度适中 | 日常检测 | 15题 |
| ME03 | 模拟卷·冲刺拔高 | 完整高考结构，难度偏难 | 冲刺阶段 | 15题 |
| ME04 | 专项卷·力学综合 | 全力学计算大题 | 专题强化 | 5题 |
| ME05 | 专项卷·电磁学综合 | 全电磁学计算大题 | 专题强化 | 5题 |

#### 试卷结构标准（以物理为例）

```
模拟试卷（完整结构）
├── 选择题（8道，每题6分 = 48分）
│   ├── 单选4道（基础2道+中档2道）
│   └── 多选4道（中档2道+较难2道）
│
├── 实验题（2道，每题8-10分 = 18分）
│   ├── 力学实验1道
│   └── 电学实验1道
│
├── 计算大题（3道 = 34分）
│   ├── 力学大题1道（中档）
│   ├── 电磁学大题1道（中偏难）
│   └── 压轴大题1道（多过程综合）
│
└── 选做题（2选1，10分）
    ├── 热学题1道
    └── 振动与波/光学题1道

总分：110分 | 建议用时：75分钟
```

#### 功能要求
- **限时模式**：开始后倒计时，到时间自动交卷
- **自动批改**：选择题和填空题自动判分，计算题按步骤给分（AI辅助或人工标准对照）
- **成绩分析**：得分率按知识板块统计，生成雷达图
- **与知识点联动**：错题自动关联到对应的模型和套路

#### 数据结构

```typescript
interface MockExam {
  id: string;                    // 如：ME01
  title: string;                 // 试卷名称
  subject: string;
  examType: 'full' | 'topic';    // 完整试卷/专项卷
  difficulty: 'basic' | 'medium' | 'advanced';
  totalScore: number;            // 总分
  timeLimit: number;             // 时间限制（分钟）
  
  // 试卷结构
  sections: {
    sectionName: string;         // 如：选择题
    questions: {
      questionId: string;        // 关联题库中的题目
      score: number;             // 分值
    }[];
  }[];
  
  // 分析
  modelQuestions: string[];      // 参照的真题年份（如基于2024全国甲卷）
  tags: string[];
}
```

## 三、高三专题页面的整体结构

### 路由：`/physics/senior`

```
高三专题页（/physics/senior）
├── 页面顶部
│   ├── 标题：物理·高考专项
│   ├── 倒计时：距离高考还有 XXX 天
│   └── 快捷入口卡片（5个类别的导航卡片）
│
├── 类别1：高考政策与命题趋势
│   ├── 文章列表（P01-P05）
│   └── 每篇可展开阅读
│
├── 类别2：高考专题突破
│   ├── 专题卡片网格（T01-T09）
│   ├── 每个卡片显示：专题名+考频+平均得分率+题目数量
│   └── 点击进入专题详情页
│
├── 类别3：应试策略与答题模板
│   ├── 策略列表（S01-S07）
│   └── 每条可展开阅读，模板部分高亮显示
│
├── 类别4：模拟考试
│   ├── 试卷列表（ME01-ME05）
│   ├── 每张试卷卡片：名称+难度+时间+得分率（做过的话）
│   └── 点击开始考试
│
└── 类别5：升学路径与院校参考
    ├── 信息列表（A01-A06）
    └── 每条可展开阅读
```

### 五科通用性说明

以上所有类别的内容结构对六科完全通用。区别仅在于：
- **类别1（政策）**：大部分内容跨科共享（如新高考政策、赋分制），只有"考纲与考查要求"按学科不同
- **类别2（专题）**：每科的题型专题不同（如数学的"导数大题"、英语的"阅读理解专项"）
- **类别3（策略）**：每科的答题模板不同，但"时间分配""分步得分"等通用
- **类别4（模考）**：每科试卷结构不同
- **类别5（升学）**：大部分跨科共享，"专业选科要求"按学科视角不同

**建议**：类别1、3、5的通用内容抽取为跨学科共享数据，类别2、4按学科独立编写。

## 四、优先级与实施建议

### 优先级

| 优先级 | 内容类别 | 理由 |
|:---|:---|:---|
| P0 | 类别1·高考政策与命题趋势 | 信息类内容，可快速产出，家长和学生都有需求 |
| P0 | 类别3·应试策略与答题模板 | 实用性最强，直接帮助"少丢分" |
| P1 | 类别2·高考专题突破 | 需要真题积累，但框架可以先定 |
| P2 | 类别4·模拟考试 | 依赖题库达到一定量级才有意义 |
| P2 | 类别5·升学路径与院校参考 | 锦上添花，数据需要每年更新维护 |

### 建议实施时机

- **P0内容**：在物理核心内容（模型详解+套路）完成第一批后即可开始
- **P1内容**：在物理题库积累到100道以上时开始
- **P2内容**：在六科内容基本完成后，或用户明确需要时再做

### 内容产出分工

```
高考政策与命题趋势（P01-P05）
├── P01 新高考改革政策解读 → AI搜集+整理
├── P02 各省高考模式一览 → AI搜集+整理
├── P03 物理学科考纲与考查要求 → AI搜集+整理
├── P04 近五年命题趋势分析 → AI分析历年真题后归纳
└── P05 新题型与考法变化 → AI分析近两年真题后归纳
```
