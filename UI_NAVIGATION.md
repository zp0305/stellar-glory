## 一、页面布局规范

### 1.1 AppLayout 布局规则

```
┌─────────────────────────────────────────────────────┐
│                    TopNav（固定顶部）                 │
├──────────┬──────────────────────────────┬────────────┤
│          │                              │ RightNav  │
│ LeftSidebar│         主内容区            │（可选）   │
│（固定）  │      max-w-3xl 居中         │ anchors   │
│ 240px    │                              │ 仅Vision  │
└──────────┴──────────────────────────────┴────────────┘
```

**AppLayout 传参规范**：

| 页面 | showSubjectNav | anchors |
|------|:---:|:---:|
| SiteHome | ❌ | — |
| PhysicsModuleGrid | ❌ | — |
| 模型列表 | ✅ | — |
| **模型详情** | ✅ | **[]**（无锚点导航） |
| **套路详情** | ✅ | **[]**（无锚点导航） |
| VisionListPage | ✅ | — |
| **VisionDetailPage** | ✅ | **✅**（章节锚点） |
| **题库列表** | ✅ | — |
| **题库详情** | ✅ | — |
| **做题页** | ✅ | — |

### 1.2 内容区宽度

- 主内容：`max-w-3xl mx-auto`
- 卡片：标准 Card + CardContent
- 公式渲染：MarkdownRenderer + enableKaTeX

## 四、样式规范

### 3.1 难度颜色

| 难度 | 颜色 | 背景 | 边框 |
|------|------|------|------|
| B 基础 | `text-green-600` | `bg-green-50` | `border-green-200` |
| J 进阶 | `text-amber-600` | `bg-amber-50` | `border-amber-200` |
| T 挑战 | `text-red-600` | `bg-red-50` | `border-red-200` |

### 3.2 选项卡片状态

| 状态 | 样式 |
|------|------|
| 未选 | `border-border bg-card` |
| 选中 | `border-primary bg-primary/5` |
| 正确（提交后）| `border-green-500 bg-green-50 text-green-700` |
| 错误（提交后）| `border-red-400 bg-red-50 text-red-600` |
