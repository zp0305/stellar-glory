## 三、题目模板

```ts
{
  id: 'PHY-B01',           // 格式：{学科}-{难度}{序号}，如 PHY-B01
  modelId: 'PHY-M01',       // 对应模型编号
  difficulty: 'B',           // B | J | T
  type: '选择题',           // 选择题 | 填空题 | 计算题
  estimatedMinutes: 8,       // B层5-10 / J层10-15 / T层15-20
  tags: ['易错点', '真实情境'],  // 1-3个，从标签库选取

  // 提示：选择题必须有提示；计算题建议有提示
  hint: '解题方向提示，不直接给答案',  // 或 null（无提示）

  // 题干：Markdown + LaTeX
  question: '【题干正文】\n\n已知条件...',

  // 选择题：4个选项；填空/计算题：null
  options: [
    'A. 选项内容',
    'B. 选项内容',
    'C. 选项内容',
    'D. 选项内容'
  ],

  answer: 'B',               // 选择题=字母；填空/计算=数值或表达式（字符串）

  // 解析：Markdown + LaTeX
  explanation: `## 解析\n\n1. 判断为[模型类型]，使用[核心公式]\n\n2. 代入数据...\n\n**答案：40m**`,

  // 关联知识点（来自模型的 knowledgeNetwork）
  points: ['匀变速直线运动', '刹车距离'],

  // 可选：关联套路编号
  routineIds: ['PHY-R02']
}
```

## 四、难度标准

### B层 · 基础题（3-4道/模型）

**特征**：单一模型，直接套公式
**时间**：5-10分钟
**标签推荐**：`公式应用`、`易错点`、`概念辨析`

### J层 · 进阶层（3-4道/模型）

**特征**：多过程分析，2个模型组合
**时间**：10-15分钟
**标签推荐**：`多过程分析`、`追及相遇`、`综合应用`

### T层 · 挑战层（1-2道/模型）

**特征**：技巧性强/综合度高
**时间**：15-20分钟
**标签推荐**：`技巧性强`、`逆向思维`、`临界分析`

## 五、题型说明

### 选择题（占比70%）

- **必须包含4个选项**（A/B/C/D）
- 选项之间互斥，不能有两个以上正确
- 正确答案分布尽量均衡（B/J/T层各2-3道）
- **必须有提示**（hint）

### 填空题（占比15%）

- 答案必须是**唯一确定的数值或表达式**
- 例：`answer: '40m'` / `answer: '20'` / `answer: '0.5s'`
- **支持自动判分**：学生输入值与 `answer` 完全一致即判对
- 可以包含1-2个小问

### 计算题（占比15%）

- **提交后显示参考解答**，学生自行对比判断对错（不自动判分）
- 解析必须写出**完整步骤**，包含公式推导和最终答案
- 建议包含1-2个技巧性步骤或巧算方法
- 答案格式：`answer: '数值+单位'`，如 `answer: '62.5m'`

## 六、标签库（从以下选取，1-3个/题）

| 标签 | 适用场景 |
|------|----------|
| `公式应用` | 直接套用核心公式即可求解 |
| `易错点` | 常见错误陷阱（刹车距离、单位换算等）|
| `真实情境` | 生活场景（汽车刹车、高铁运行等）|
| `多过程分析` | 加速→匀速→减速等多阶段 |
| `追及相遇` | 两物体相对运动 |
| `综合应用` | 多个模型、多个公式组合 |
| `逆向思维` | 倒推或从结果反推条件 |
| `技巧性强` | 有特殊解法或二级结论 |
| `概念辨析` | 考察物理概念理解 |
| `图像识别` | 从 v-t、x-t 图中获取信息 |
| `临界条件` | 恰好到达、最高点、临界状态 |
| `多物体分析` | 涉及多个研究对象 |

## 七、LaTeX 公式规范

| 内容 | 写法 | 示例 |
|------|------|------|
| 行内公式 | `$公式$` | $v = v_0 + at$ |
| 独立公式 | `$$公式$$` | $$\bar{v} = \frac{v_0 + v}{2}$$ |
| 下标 | `_{}` | $v_0$、$S_1$ |
| 分数 | `\frac{分子}{分母}` | $\frac{v^2 - v_0^2}{2a}$ |
| 开方 | `\sqrt{}` | $\sqrt{2gh}$ |
| 角度 | `^\circ` | $30^\circ$ |
| 乘方 | `^2` | $10^2$ |

**注意**：数字与单位之间加空格：`20m/s` → `$20m/s$`

## 八、解析模板

```ts
explanation: `## 解析

**第一步：判断模型与公式**
判断为刹车模型，末速度 v=0，代入速度-位移公式。

**第二步：代入数据**
$$v^2 - v_0^2 = 2aS$$
$$0 - 400 = 2 \times (-5) \times S$$

**第三步：计算结果**
$$S = \frac{400}{10} = 40m$$

**答案：40m**

> 💡 技巧：刹车距离也可直接用 $S = \frac{v_0^2}{2a}$ 一步得出。`
```
